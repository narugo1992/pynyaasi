Installation
===================

``pynyaasi`` is currently hosted on PyPI. It required python >= 3.8.

You can simply install pynyaasi from PyPI with the following command:

.. code:: shell

    pip install pynyaasi

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/narugo1992/pynyaasi.git@main

of ``pynyaasi`` should be shown.
